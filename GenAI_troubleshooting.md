### Use of Gen AI in the project

M365 Microsoft Copilot (with mandatory enterprise data protection enabled)
https://m365.cloud.microsoft/

Copilot was used to find help on specific Azure and Kubernetes commands also the install for UERANSIM and iPerf3.

I initially tried to use the inbuild Copilot in Azure's portal when getting quota issues trying to set up the AKS Cluster, however this was not successful and I switched to use the Azure CLI and putting individual errors into Copilot, which was successful.

The initial YAML file and command line for Helm install of Open5gs was retrieved from Copilot
