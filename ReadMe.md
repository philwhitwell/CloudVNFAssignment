The project was started in PyCharm as I was anticipating the need to automate the load testing using Python. However the project is organised as follows

The project is divided into 3 folders:
    1. The first folder is for the collection of evidence.
    2. The second folder is the bash scripts used to install and run the load testing.
    3. The third folder is a collection of the yaml configuration files used.

The following markdown files are used to organise the project:
    1. The first markdown file is the GenAI troubleshooting.md file.
    2. The second markdown file is the main queries that were used in Prometheus, as early developed queries were lost, there is also two dashboard extracts (once I learnt this was possible)
    3. The third markdown file is the collation of the notes for the answers needed in Gradescope.
